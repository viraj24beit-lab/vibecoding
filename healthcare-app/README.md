## HealthCare Hub

A simple healthcare web app scaffold built with React, Vite, TypeScript, React Router, and Tailwind CSS v4.

### Features
- Home page with hero and featured services
- Services, Doctors, Appointments, and Contact pages
- Appointment form with basic client-side validation
- Responsive UI styled with Tailwind CSS

### Getting started

1. Install dependencies:
```bash
npm install
```
2. Start the dev server:
```bash
npm run dev
```
3. Build for production:
```bash
npm run build
```
4. Preview the production build:
```bash
npm run preview
```

### Tailwind CSS

This project uses Tailwind CSS v4. Styles are enabled via `@import "tailwindcss";` in `src/index.css`. No Tailwind config is required for the basics; add a `tailwind.config.ts` if you need customization.
